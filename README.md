## 기능 설명

### [성게를 피해 물고기를 클릭하는 게임]

- 주어진 시간안에 성게를 피해 물고기를 클릭하는 게임입니다.

```
- ▶버튼을 누르면 게임이 시작되고 시작한 이후에 ⏸버튼을 누르면 게임이 일시정지 됩니다.
- 일시정지한 이후에 나오는 팝업에 🔄버튼을 누르면 처음부터 재시작됩니다.
- 시간이 0초가 될때까지 당근을 모두 클릭하지 못하면 패배합니다.
```
